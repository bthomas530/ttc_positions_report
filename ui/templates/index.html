IBKR Trade History (Flex Query)

Tranche and income tracking are built from your IBKR trade history, imported automatically once a day through IBKR's Flex Web Service. This needs a one-time setup in IBKR Client Portal (steps below). The token is stored only on this computer.

How to set this up in IBKR (one time, about 5 minutes)
  1. Log in to IBKR Client Portal at www.interactivebrokers.com (the website, not Trader Workstation).
  2. Open the menu and go to Performance & Reports → Flex Queries.
  3. Under Activity Flex Query, click the + to create a new query.
  4. Name it TTC Trades. Under Sections, pick Trades and check: Executions; select all fields (or at least: Symbol, Underlying Symbol, Asset Class, Buy/Sell, Quantity, Trade Price, Proceeds, Commission, Trade Date, Date/Time, Open/Close Indicator, Notes/Codes, Strike, Expiry, Put/Call, Multiplier, Trade ID, Order ID).
  5. Under Delivery Configuration: Period = Last 365 Calendar Days, Format = XML.
  6. Save the query and note the Query ID number shown in the list — that goes in the box above.
  7. Now go to Settings → Account Settings, find Flex Web Service, turn it on, and generate a token. Copy the long token string into the box above.
  8. Click Save, then Import now. Done — from now on it imports by itself.
